# deepzoom.py: Python Deep Zoom Tools

## Installation

```bash
git clone https://github.com/openzoom/deepzoom.py.git
cd deepzoom.py
python setup.py install
```

## Example

```bash
cd examples/helloworld/

# Single image (DZI)
./helloworld-dzi.py

# Collection (DZC)
./helloworld-dzi.py
```
